Hello there! I see that you're currently viewing this project :-)

Here are the following functionalities of this app for your own reference:

1. Create a task

2. Show all the tasks you created

3. You can edit or update the task in 2 ways:
    a. 1st option: Click on the >> Pencil Icon << to edit the task
    b. 2nd option: Double click the text to edit the task. Once done, you can hit >> Enter key on your keyboard << to save the update/s

4. Delete a task

5. Click the checkbox to indicate a completed task
